Cara menambahkan Shortcut Key untuk Koneksi Internet Anda

Dengan cara ini dasar dan umum untuk dial koneksi internet dengan menggunakan Dial Up :

cara 1) Klik pada Start> Pengaturan Button lalu klik pada Network dan dial-up opsi koneksi, di sini anda dapat dial koneksi internet Anda.

Cara 2)untuk dial koneksi internet Anda jika Anda memiliki cara pintas untuk koneksi internet Anda di desktop Anda. Cukup boot sistem dan dial koneksi internet Anda dari shortcut desktop Anda.

cara3) untuk dial koneksi internet dengan dengan menggunakan mouse, cukup tambahkan tombol shortcut yang akan memungkinkan Anda untuk melakukan hal ini. 7. sebutkan 5 jenis koneksi internet dan bagaimana cara/langkah menghubungkan perangkat komputer kita agar terkoneksi dg internet​


Jawaban:

1.) Dial-Up

2.) ADSL

3.) LAN

4.) TV Kabel

5.) WiFi

Penjelasan:

1.) Dial-Up

Jika kamu tumbuh besar di era awal internet masuk ke Indonesia, maka koneksi dial-up bukan suatu teknologi yang asing. Dial-up mengandalkan jaringan telepon rumah berbentuk kabel. Pada masanya, kamu bisa melepas kabel jaringan yang terpasang di pesawat telepon lalu menyambungkannya ke komputer di rumah.

Dial-up ini akan bekerja melalui jalur PSTN (Public Switched Telephone Network) hingga bisa terhubung dengan ISP (Internet Service Provider). Bisa dibilang dial-up ini sangat membekas di dalam sejarah teknologi internet Indonesia. Walaupun dari kecepatan yang ditawarkan tidak tinggi, tapi sudah cukup untuk menjelajah dunia maya pada masanya.

2.) ADSL

ADSL (Asymmetric Digital Subscriber Line) adalah teknologi modem yang bekerja dalam frekuensi 34 kHz hingga 1104 kHz. Dengan ADSL, maka kamu bisa mengirimkan suara hingga fax ke pengguna lainnya. Dengan harga berlangganan lebih ekonomis, kecepatan pengiriman data tetap terpengaruh dari jarak komputer kamu dengan gardu telepon.

3.) LAN

LAN termasuk koneksi internet yang banyak dikenal saat ini. Sistem kerjanya adalah menggunakan satu komputer sebagai server yang terhubung dengan internet dengan kabel telepon. Lalu komputer lainnya baru bisa terhubung internet. Merupakan akronim dari Local Area Network, jaringan koneksi internet ini hanya mencakup wilayah yang sangat kecil.

4.) TV Kabel

TV kabel pada saat ini juga telah menggunakan koneksi internet. Jaringan TV kabel mampu mencapai kecepatan hingga 27 Mbps. Selain lebih cepat, kelebihan lain dari TV kabel adalah biaya iuran tetap setiap bulan, tidak ada jam sibuk, hingga layanan tanpa batas. Namun ada kekurangannya juga. Selain dari sisi harga lebih tinggi, tidak semua wilayah telah mendapatkan jaringan ini.

18. cara mengkoneksikan mikrotik ke internet


1. login ke mikrotik via winbox -> masuk ke menu IP -> DHCP Client -> tambahkan DHCP Client dngan mengklik menu +
- Interface : plih interface yg trhubung ke sumber internet
- Use Peer DNS : centang
- Use Peer NTP : centang
- Add Default Router : Yes

2. setting IP Address. masuk ke menu IP -> Addresses -> tambahkan IP Address untk inferface yg menuju ke client dengan IP Address 10.10.10.10/24. interface NAT sdah mendapatkan IP Address scara otomatis krena sebagai DHCP Client dri sumber internet.

3. tambahkan setingan DNS Server. msuk ke menu IP -> DNS. krena DHCP Client pada kolom "Use Peer DNS" dicentang, mka mikrotik sdah dpat DNS Server scara otomatis. tpi untk cadangan bsa tambahkan lagi pakai DNSnya google : 8.8.8.8

4. tambahkan setingan NAT Masquerade. masuk ke menu IP -> Firewall -> masuk tab NAT -> tambahkan rule NAT :
~ Tab General
- Chain : srcnat
- Out-Interface : plih interface yg menuju ke internet
~ Tab Action
- Action : Masquerade

5. skrang mikrotik harusnya sdah konek ke internet. cek koneksi dngan lakukan PING via terminal

29. bagaimana cara kerja koneksi internet menggunakan koneksi LAN


Internet sendiri merupakan singkatan dari interconnection-networking yang artinya adalah jaringan komputer yang dapat menghubungkan seseorang dengan dunia (telekomunikasi) dengan menggunakan standart sistem global transmission control protocol/ Internet Protocol Suite (TCP/IP), yang berperan sebagai protocol pertukaran paket (Packet Switching Communication Protocol) untuk melayani seluruh pengguna internet yang terdapat di dunia.

Namun perlu diketahui bahwa internet juga memiliki beberapa jenis yang diantaranya bernama LAN (Local Area Network), MAN (Metropolitan Area Network) dan WAN (Wide Area Network).

Berikut ini adalah pengertian dari hal-hal tersebut, serta bagaimana cara kerjanya untuk dapat terhubung dengan internet.
