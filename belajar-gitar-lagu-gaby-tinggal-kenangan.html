34. Quiss terakhir perjumpaan Banayk susunan kata dari -Tinggal -Kenangan Note: Tinggal kenangan bersama akun nih


TinggalTotal = 7Unsur Ganda = 2(g)

Banyak susunan~

= 7! / 2!

= 7.6.5.4.3.2 / 2

= 5.040 / 2

= 2.520 susunan

Kenangan

Total = 8

Unsur Ganda = 3(n), 2(a)

Banyak susunan~

= 8! / 3!.2!

= 8.7.6.5.4.3.2 / 3.2.2

= 40.320 / 12

= 3.360 susunan

Jawaban:

TINGGAL

Jumlah Unsur = 7!

Unsur Ganda = 2!

P = n! / k!

P = 7! / 2!

P = (7×6×5×4×3×2×1) / (2×1)

P = 5.040 / 2

P = 2.520 Susunan Kata✅

KENANGAN

Jumlah Unsur = 8!

Unsur Ganda = 3!, 2! = 12

P = n! / k!

P = 8! / 12

P = (8×7×6×5×4×3×2×1) / 12

P = 40.320 / 12

P = 3.360 Susunan Kata✅
